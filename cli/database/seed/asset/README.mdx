# Asset

This directory contains the asset seed files for the Vircadia World.

## Environment Variables

- <code>VRCA_CLI_SERVICE_POSTGRES_SEED_SYSTEM_ASSET_DIR</code>: Overrides the default seed files (as seen below) with a custom set of files
- <code>VRCA_CLI_SERVICE_POSTGRES_SEED_USER_ASSET_DIR</code>: Optional: sets another directory for user-defined seed files
