# Client Overview

The Vircadia World client is responsible for connecting to the Vircadia World server and rendering the virtual world.

## Available Clients

Vircadia World supports the following client types:

- Web Babylon.js

## Connecting to the Server

To connect to a Vircadia World server, clients need to use both REST and WebSocket APIs. For detailed information on available endpoints and connection methods, refer to the [API Reference](/vircadia-world/server/api-reference) documentation.
