# Seed

## Asset

The `cli/database/seed/asset` directory contains the asset seed files for the Vircadia World.

### Environment Variables

- <code>VRCA_CLI_SERVICE_POSTGRES_SEED_SYSTEM_ASSET_DIR</code>: Overrides the default seed files (as seen below) with a custom set of files
- <code>VRCA_CLI_SERVICE_POSTGRES_SEED_USER_ASSET_DIR</code>: Optional: sets another directory for user-defined seed files

### Asset Seed Files

1. `babylon_levelTest.glb` - A Babylon.js level test asset

## SQL

The `cli/database/seed/sql` directory contains the SQL seed files for the Vircadia World.

### Environment Variables

- <code>VRCA_CLI_SERVICE_POSTGRES_SEED_SYSTEM_SQL_DIR</code>: Overrides the default seed files (as seen below) with a custom set of files
- <code>VRCA_CLI_SERVICE_POSTGRES_SEED_USER_SQL_DIR</code>: Optional: sets another directory for user-defined seed files

### SQL Seed Files

1. `babylon_levelTest.sql` - A Babylon.js level test
